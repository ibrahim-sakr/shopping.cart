## Shopping Cart

the [source](https://code.tutsplus.com/tutorials/create-a-simple-shopping-cart-using-angularjs-part-1--cms-23535)

this is a simple shopping cart powerd by [angularJS](https://angularjs.org/) and [NODEJS](https://nodejs.org/)

the main porpose is learning

## Install

you need [bower](http://bower.io/) to install Frontend dependencies

so after installing bower run `bower install`

and [NPM](https://www.npmjs.com/) to install backend dependencies

so after install nodejs run `npm install`
